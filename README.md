# User Copy Script

## Introduction
Welcome to my first GitHub project! This user copy script has been created to automate common tasks on Windows. It utilizes batch scripting to simplify and expedite the process of copying files and folders.

## Features
The script provides the following features:
- Copying browser bookmarks (Google Chrome, Microsoft Edge).
- Copying Outlook signatures.
- Copying Outlook templates.
- Copying desktop background.

## Usage
The script uses default locations for files and folders on a Windows system. It performs the copy to the C: drive in a folder named "Fav_User". Make sure you have the necessary permissions.

## Getting Started
To use the script:
1. Download the script file (`script.bat`) to your local machine. ( It is possible to use the script from a USB key )
2. Run the script by double-clicking on it or executing it from the command line.

Feel free to customize the script based on your specific needs and configurations.
